# markdown写作常见语法

> 针对偏技术文档、教程类写作，`markdown`可读性高，语法学习成本也低，非程序员也可以快速掌握，最近刚接触obsidian用于实践个人第二大脑知识管理，以`markdown`格式写一篇应用语法的笔记作为教程

你是否也曾有过这样的瞬间：

> - 灵感乍现，几分钟后就忘得一干二净？
> - 读过一本好书，当时醍醐灌顶，一个月后却只记得书名？
> - 信息爆炸，收藏了无数文章，却从未打开回顾？

我们生活在一个信息过载的时代，而我们的大脑，天生就不是为了“存储”而是为了“思考”。试图用大脑记住所有东西，就像用CPU当硬盘用，不仅低效，而且会拖慢整个系统的运行速度。是时候，为你的思想，建造一座外部的、可靠的、可无限扩展的「第二大脑」了。

## 💡 Markdown 语法解析 1：标题与引用
- `#` 号代表一级标题，是文章的标题。
- `##` 代表二级标题，用于划分大的章节。
- `>` 符号用于创建引用块，非常适合用来引出问题、引用名言或突出重点。引用块内部还可以嵌套其他语法，比如这里的列表。

## 什么是「第二大脑」？
「第二大脑」并非什么黑科技，它本质上是一个**个人知识管理系统**。它是一个由你亲手打造、完全为你服务的数字知识库，用来捕捉你所有的想法、学习和灵感。

## 💡 Markdown 语法解析 2：图片
- 图片的语法是 `![图片的替代文字](图片链接)`。
- `![...]` 内的文字是“替代文字”，当图片加载失败时会显示，也对搜索引擎和视觉障碍用户友好。
- `(...)` 内是图片的URL。在公众号中，这里可以先放网络图片，粘贴到编辑器后，系统会提示你替换为本地已上传的图片。
- obsidian 使用`![[图片链接]]`

## 为什么你需要它？从“存储”到“连接”
传统的笔记方式，更像是一个“数字仓库”，东西堆进去就再也想不起来了。而一个真正的第二大脑，核心在于**“连接”**


## 💡 Markdown 语法解析 3：表格
- 表格用 `|` 来分隔列，用 `-` 来分隔表头和表体。
- `:---` 表示左对齐，`:--:` 表示居中对齐，`---:` 表示右对齐。合理使用对齐方式能让表格更美观。
```

| 传统笔记      | 第二大脑      |
| :-------- | :-------- |
| 目标是“存下来”  | 目标是“用起来”  |
| 信息孤岛，难以检索 | 网状结构，相互关联 |
| 被动收集，遗忘曲线 | 主动创造，复利增长 |
```

## 如何开始：四步打造你的第二大脑
构建第二大脑的方法有很多，比如著名的 `PARA` 方法、`Zettelkasten`（卡片盒笔记法）。但无论哪种，其核心都离不开以下四个步骤：
### 1. 捕捉：像海绵一样吸收
养成随时记录的习惯。无论是读到的一段话、一个突发的想法，还是一次会议的结论，**立刻捕捉**下来。
- **工具推荐**：`Obsidian`, `Notion`, `Flomo`, 甚至系统自带的备忘录。
- **关键**：**降低记录门槛**，让记录动作比“思考要不要记”更快。
### 2. 处理：给思想“去粗取精”
定期回顾你捕捉到的信息。这是最关键的一步。问自己几个问题：
- 这个信息是什么意思？（用自己的话复述）
- 它为什么重要？
- 它能和哪些我已知的知识联系起来？
- 我该如何行动？
对于重要的信息，将其提炼成一张张独立的、标准化的“知识卡片”。

## 💡 Markdown 语法解析 4：列表与行内代码
- `-` 或 `*` 开头是无序列表。
- `1.` 开头是有序列表。
- `` `Obsidian` `` 这样用反引号包裹的内容，是“行内代码”，非常适合用来突出工具名、文件名、专有名词等。
---
下面是一个典型的知识卡片示例，它记录了关于“原子化输入”的理解：

```markdown
## 原子化输入

原子化输入是一种写作理念，它鼓励我们将庞大、模糊的想法，拆解成一个个最小、最独立的“思维原子”进行记录。 
**核心价值**：

- 降低写作启动的心理门槛。
- 便于后续的重组和连接，产生新的创意。
**应用场景**：

- 写作前，用关键词或短句记录所有相关想法。
- 阅读时，将核心观点提炼为一句话。

#主题：认知写作
#标签：#写作方法 #认知科学
```


---
## 💡 Markdown 语法解析 5：代码块

三个反引号包裹的内容是代码块,后面可以加上语言名称（如 `markdown`），许多编辑器会为此提供语法高亮，让代码更易读。
    ```markdown
    包裹的代码块
    ```
- 代码块非常适合用来展示示例、配置文件或任何需要保持原始格式的文本。

### 3. 连接：编织你的思想网络

当你的知识卡片越来越多，真正的魔力就发生了——**连接**。
- 在一张卡片中，通过 `[[双链]]` 引用另一张卡片。
- 为卡片打上 `#标签`，形成主题聚合。
- 使用图谱视图，直观地看到你的知识网络。

> “创造力，不过是把事物连接起来的能力。” —— 史蒂夫·乔布斯
你的第二大脑不再是一个线性的文件夹，而是一个充满活力的、可以自行生长的**思想生态系统**。
### 4. 创造：从输入到输出

建造第二大脑的最终目的，不是为了成为一个“收藏家”，而是为了成为一个“创造者”。

**当你需要写一篇文章、准备一个演讲、解决一个复杂问题时，你不再是面对一个空白文档，而是进入你丰富的思想网络，从中提取、重组、连接，最终创造出属于你自己的、独一无二的成果。**
